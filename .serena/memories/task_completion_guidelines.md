# タスク完了時のガイドライン

## 変更後の確認事項

### 1. ブラウザテスト
- 変更したファイルの構文エラーがないか確認
- ブラウザのコンソールでエラーをチェック
- 機能が正常に動作するか確認

### 2. 変換機能のテスト
- Python → 共通テスト変換の確認
- 共通テスト → Python変換の確認
- 演算子変換の確認
- 制御構造の変換確認

### 3. 実行機能のテスト
- Python実行の確認
- 共通テスト実行の確認
- 入力機能の確認
- 出力表示の確認

### 4. UI機能のテスト
- エディタの動作確認
- ボタン機能の確認
- コピー機能の確認
- サンプル読み込みの確認

## コード品質確認

### JavaScriptファイル
```bash
# 構文チェック（Node.js環境の場合）
node -c assets/js/modules/converter.js
node -c assets/js/modules/executor.js
node -c assets/js/modules/flowchart.js
node -c assets/js/modules/ui.js
```

### ブラウザコンソール確認
- F12でコンソールを開き、エラーがないか確認
- 変換・実行時にエラーが出ないか確認

## デプロイ前確認

### ファイル整合性
- 全ての必要ファイルがコミットされているか

- コメントが適切か
