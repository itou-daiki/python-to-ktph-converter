# Python ⇄ 共通テスト用プログラム表記 変換ツール - プロジェクト概要

## プロジェクトの目的
このプロジェクトは、Python言語のコードと日本の大学入学共通テストで使用される「共通テスト用プログラム表記」を相互に変換し、実行できるWebアプリケーションです。

## 技術スタック

### フロントエンド
- HTML5, CSS3, JavaScript (ES6+)
- エディタ: CodeMirror 5.65.13
- フローチャート生成: Mermaid.js 10.6.1

### 実行環境
- Python実行: Pyodide 0.24.1（ブラウザ内でPython実行）
- 共通テスト用プログラム実行: カスタムインタープリター

### アーキテクチャ
- モジュラー設計
- GitHub Pages対応
- レスポンシブデザイン

## 主な機能

### 双方向変換
- Python → 共通テスト用プログラム表記
- 共通テスト用プログラム表記 → Python

### コード実行機能
- ブラウザ内でPythonコード実行
- 共通テスト用プログラム表記のカスタムインタープリター実行

### その他の機能
- フローチャート自動生成
- URL共有機能
- サンプルコード読み込み
- コピー機能

## 対応する変換規則
- 基本入出力: `print()` ↔ `表示する()`, `input()` ↔ `【外部からの入力】`
- 制御構造: `if/elif/else` ↔ `もし/そうでなくもし/そうでなければ`
- 繰り返し: `while` ↔ `の間繰り返す`, `for range()` ↔ `を...から...まで...ずつ増やしながら繰り返す`
- 演算子: `//` ↔ `÷`, `%` ↔ `％`
- 関数: `len()` ↔ `要素数()`, `int()` ↔ `整数()`, `str()` ↔ `文字列()`

## プロジェクト構造
- `index.html`: メインHTMLファイル
- `assets/js/`: JavaScriptコード
  - `app.js`: メインアプリケーション
  - `modules/`: 機能別モジュール
    - `converter.js`: 変換ロジック
    - `executor.js`: 実行エンジン
    - `flowchart.js`: フローチャート生成
    - `ui.js`: UI管理
- `assets/css/`: スタイルシート
- `Sample/`: サンプルコード
