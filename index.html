🐍 Python記述時の注意点

記号は半角で入力

= : ( ) [ ] { }
+ - * / %
== != < > <= >=
全角文字を使うとエラーになります

文字列は「'」や「"」で囲う

'文字列' または "文字列"
全角の記号’、”はエラーになります

インデントは「タブ」

タブや半角スペース4つでインデント
if文、for文、def文の中身は必ずインデントが必要です

コロン（:）を忘れずに

for文、while文
if文、def文
の末尾には:が必要
セミコロン;ではないので注意
