
1. 基本構造
順次構造
出力文

Python: print("文字列")

入力文

Python: 変数名 = input()
共通テスト用: 変数名 = 【外部からの入力】

分岐構造
if文

Python: if 条件:
共通テスト用: もし 条件 ならば:

elif文

Python: elif 条件:
共通テスト用: そうでなくもし 条件 ならば:

else文

Python: else:
共通テスト用: そうでなければ:

繰り返し構造
while文

Python: while 条件:
共通テスト用: 条件 の間繰り返す:

for文（range使用）

Python: for 変数 in range(開始値, 終了値, 増分):
共通テスト用: 変数 を 開始値 から 終了値-1 まで 増分 ずつ増やしながら繰り返す:

2. 演算子の変換
算術演算子

加算: + → +（そのまま）
減算: - → -（そのまま）
乗算: * → *（そのまま）
除算: / → /（そのまま）
整数除算: // → ÷（商の整数値）
剰余: % → ％
べき乗: ** → **（そのまま）

例：
Python: result = a / b
共通テスト用: result = a / b

Python: quotient = a // b
共通テスト用: quotient = a ÷ b  #演算子÷は商の整数値を返す

Python: remainder = a % b
共通テスト用: remainder = a ％ b
比較演算子

等しい: == → ==（そのまま）
等しくない: != → !=（そのまま）
より小さい: < → <（そのまま）
以下: <= → <=（そのまま）
より大きい: > → >（そのまま）
以上: >= → >=（そのまま）

論理演算子

論理積: and → and（そのまま）
論理和: or → or（そのまま）
否定: not → not（そのまま）

例：
Python: if x > 0 and y < 10:
共通テスト用: もし x > 0 and y < 10 ならば:

4. 配列操作
配列の定義

Python: 配列名 = [要素1, 要素2, 要素3]
共通テスト用: 配列名 = [要素1, 要素2, 要素3]（配列名は先頭文字が大文字）

Python: 配列名[インデックス]（0から開始）
共通テスト用: 配列名[インデックス]（特に説明がない場合は0から開始）

「配列名のすべての値を0にする」のような表記も可能

5. 関数定義と呼び出し
関数の定義

共通テスト用: 関数定義の明示的な記法はない（問題文で説明される）

共通テスト用: 値を返す関数として定義される

関数の呼び出し

Python: 結果 = 関数名(引数1, 引数2)
共通テスト用: 結果 = 関数名(引数1, 引数2)

要素数(配列): 配列の要素数を返す
整数(値): 値を整数に変換
文字列(値): 値を文字列に変換
乱数(): 0以上1未満のランダムな値を返す

Python:

もし score >= 80 ならば:

9. 変換時の重要な注意点

除算記号: 通常の除算は / のまま、整数除算（//）のみ ÷ に変換
文字列連結: + 演算子をそのまま使用可能（PDFで明記）
表示する関数: カンマ区切りで文字列や数値を連結できる
比較演算子: すべてそのまま（==, !=, <=, >= など）
論理演算子: and, or, not をそのまま使用
剰余演算: % は ％ に変換
制御記号: 途中は ｜、最後は ⎿（特殊な縦棒とL字型記号）
関数名: 組み込み関数は日本語（表示する、要素数、整数、乱数等）
配列名: 先頭文字を大文字にする
配列インデックス: 特に説明がない場合は0から開始
範囲指定: range(開始, 終了) は「開始から終了-1まで」
減らしながら: 変数 を 開始値 から 終了値 まで 減分 ずつ減らしながら繰り返す:も可能

10. 完全な変換例
Python版
python# Python版 - 二分探索
data = [3, 18, 29, 33, 48, 52, 62, 77, 89, 97]
kazu = len(data)
atai = int(input('0～99の数字を入力してください'))
hidari = 0
migi = kazu - 1
owari = 0

while hidari <= migi and owari == 0:
    aida = (hidari + migi) // 2
    if data[aida] == atai:
        print(atai, 'は', aida, '番目にありました')
        owari = 1
    elif data[aida] < atai:
        hidari = aida + 1
    else:
        migi = aida - 1

if owari == 0:
    print(atai, 'は見つかりませんでした')

print('添字', ' ', '要素')
for i in range(0, kazu, 1):
    print(i, ' ', data[i])
