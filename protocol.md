Pythonから共通テストプログラム表記への変換プロトコル
1. 順次構造
出力文

Python: print("文字列")
共通テスト用: 表示する("文字列")

例：
Python: print("おはよう")
共通テスト用: 表示する("おはよう")
2. 分岐構造
if文

Python: if 条件:
共通テスト用: もし 条件 ならば:

elif文

Python: elif 条件:
共通テスト用: そうでなくもし 条件 ならば:

else文

Python: else:
共通テスト用: そうでなければ:

制御範囲の表現

Python: インデントで制御範囲を表現
共通テスト用: ┃、┣、┗ 記号で制御範囲と分岐構造を表現

例：
Python:
if kuji <= 4:
    print("大吉")
elif kuji <= 7:
    print("中吉")
else:
    print("小吉")

共通テスト用:
もし kuji <= 4 ならば:
┣ 表示する("大吉")
そうでなくもし kuji <= 7 ならば:
┣ 表示する("中吉")
そうでなければ:
┗ 表示する("小吉")
3. 繰り返し構造
while文

Python: while 条件:
共通テスト用: 〜の間繰り返す:

例：
Python:
i = 1
while i <= 3:
    print(i)
    i = i + 1

共通テスト用:
i = 1
i <= 3 の間繰り返す:
┃ 表示する(i)
┗ i = i + 1
for文（range使用）

Python: for 変数 in range(開始値, 終了値, 増分):
共通テスト用: 〈変数〉を〈初期値〉から〈終了値〉まで〈増分〉ずつ増やしながら繰り返す:

例：
Python:
for i in range(1, 3+1, 1):
    print(i)

共通テスト用:
iを1から3まで1ずつ増やしながら繰り返す:
┗ 表示する(i)
4. 配列の定義と操作
一次元配列

Python: 配列名 = ["要素1", "要素2", "要素3"]
共通テスト用: 配列名 = ["要素1", "要素2", "要素3"]

配列要素のアクセス

Python: print(配列名[インデックス])
共通テスト用: 表示する(配列名[インデックス])

配列の要素数取得

Python: len(配列名)
共通テスト用: 要素数(配列名)

例：
Python:
data = ["りんご", "バナナ", "みかん"]
print(len(data))

共通テスト用:
data = ["りんご", "バナナ", "みかん"]
表示する(要素数(data))
二次元配列

Python: 配列名 = [["要素"], ["要素"]]
共通テスト用: 配列名 = [["要素"], ["要素"]]

5. ユーザ定義関数
関数の定義

Python: def 関数名(引数1, 引数2):
共通テスト用: 関数名(引数1, 引数2):

戻り値

Python: return 戻り値
共通テスト用: 〜を返す

例：
Python:
def keisan(atai1, atai2):
    kekka = atai1 * atai2 / 2
    return kekka

共通テスト用:
keisan(atai1, atai2):
┃ kekka = atai1 * atai2 / 2
┗ kekkaを返す
制御範囲記号の使い分け

┃: 制御範囲の継続を表す（途中の処理）
┣: 分岐構造で、まだ後続の分岐がある場合
┗: 制御範囲の最後や、分岐構造の最終分岐を表す

変換時の注意点

制御構造の表現: Pythonのインデントは共通テスト用では ┃、┣、┗ 記号で表現する
キーワードの日本語化: if→もし、else→そうでなければ、while→〜の間繰り返す等
出力文: print() → 表示する()
戻り値: return → 〜を返す
配列の要素数: len() → 要素数()
配列名: 共通テスト用では1文字目を大文字にする場合がある
記号の使い分け: 処理の位置や分岐構造に応じて適切な記号を選択する
