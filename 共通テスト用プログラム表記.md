# 共通テスト用プログラム表記の例示

高等学校の「情報Ⅰ」の授業で使用するプログラミング言語は多様であることから、共通テスト『情報Ⅰ』の試作問題作成にあたり、共通テスト用のプログラム表記を使用します。以下、参考のためにその基本を例示します。

**注意事項：**
- 問題文の記述を簡潔にするなどの理由で、この説明文書の記述内容に従わない形式で出題することもあります
- 共通テスト『情報Ⅰ』の受験に際しては、当該問題文の中の説明や指示に注意し、それらに沿って解答してください
- 経過措置問題『旧情報（仮）』についても同様に扱うこととします

## 1. 変数

### 通常の変数
- 例：`kosu`、`kingaku_kei`
- 変数名は英字で始まる英数字と「_」の並び

### 配列変数
- 例：`Tokuten[3]`、`Data[2,4]`
- 配列名は先頭文字が大文字
- **特に説明がない場合、配列の要素を指定する添字は0から始まる**

## 2. 文字列

文字列はダブルクォーテーション（"）で囲む

```
moji = "I'll be back."
message = "祇園精舎の" + "鐘の声"  # +で連結できる
```

## 3. 代入文

```
kosu = 3, kingaku = 300  # 複数文を1行で表記できる
kingaku_goukei = kingaku * kosu
namae = "Komaba"
Data = [10,20,30,40,50,60]
Tokutenのすべての値を0にする
nyuryoku = 【外部からの入力】
```

## 4. 算術演算

- 加減乗除の四則演算：`+`、`-`、`*`、`/`
- 整数の除算：
  - 商（整数）：`÷`
  - 余り：`％`
- べき乗：`**`

## 5. 比較演算

- `==`（等しい）
- `!=`（等しくない）
- `>`、`<`、`>=`、`<=`

## 6. 論理演算

- `and`（論理積）
- `or`（論理和）
- `not`（否定）

## 7. 関数

### 値を返す関数の例
```
kazu = 要素数(Data)
saikoro = 整数(乱数()*6)+1
```

### 値を返さない関数の例
```
表示する(Data)
表示する(Kamoku[i],"の得点は",Tensu[i],"です")
```

**注意：**
- 「表示する」関数はカンマ区切りで文字列や数値を連結できる
- 「表示する」関数以外は基本的に問題中に説明あり

## 8. 制御文（条件分岐）

```
もし x < 3 ならば:
｜ x = x + 1
⎿ y = y + 1

もし x >= 3 ならば:
｜ x = x - 1
そうでなくもし x < 0 ならば:
｜ x = x * 2
そうでなければ:
⎿ y = y * 2

もし x == 3 ならば:
｜ x = x - 1
そうでなければ:
⎿ y = y * 2
```

**注意：** `｜`と`⎿`で制御範囲を表し、`⎿`は制御文の終わりを示す

## 9. 制御文（繰返し）

### for文形式
```
x を 0 から 9 まで 1 ずつ増やしながら繰り返す:
⎿ goukei = goukei + Data[x]
```
**注意：** 「減らしながら」もある

### while文形式
```
n < 10 の間繰り返す:
｜ goukei = goukei + n
⎿ n = n + 1
```

**注意：** `｜`と`⎿`で制御範囲を表し、`⎿`は制御文の終わりを示す

## 10. コメント

```
atai = 乱数()  #0以上1未満のランダムな小数をataiに代入する
```

**注意：** 1行内において`#`以降の記述は処理の対象とならない

# Python3との対比例

## Python3記述例

```python
data=[3,18,29,33,48,52,62,77,89,97]
kazu=len(data)
atai=int(input('0～99の数字を入力してください'))
hidari=0;migi=kazu-1
owari=0
while hidari <= migi and owari==0:
    aida=(hidari+migi)//2  #演算子//は商の整数部分
    if data[aida]==atai:
        print(atai,'は',aida,'番目にありました')
        owari=1
    elif data[aida]<atai:
        hidari=aida+1
    else:
        migi=aida-1
if owari==0:
    print(atai,'は見つかりませんでした')
print('添字',' ','要素')
for i in range(0,kazu,1):
    print(i,' ',data[i])
```
