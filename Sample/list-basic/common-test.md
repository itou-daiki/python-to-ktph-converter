# 配列の基本

配列の作成と要素の操作

```
# 配列操作のサンプル
Numbers = [10, 20, 30, 40, 50]
表示する("配列の内容:", numbers)
表示する("配列の長さ:", 要素数(numbers))

表示する("\n各要素を表示:")
i を 0 から 要素数(numbers)-1 まで 1 ずつ増やしながら繰り返す:
⎿ 表示する("numbers[" + 文字列(i) + "] = " + 文字列(numbers[i]))

表示する("\n要素の合計を計算:")
total = 0
i を 0 から 要素数(Numbers)-1 まで 1 ずつ増やしながら繰り返す:
⎿ total = total + Numbers[i]
表示する("合計:", total)
```
