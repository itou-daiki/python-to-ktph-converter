学習ポイント: 探索、番兵値、break

```

Data = [5, 2, 8, 1, 9, 3]
target = 【外部からの入力】
position = -1

i を 0 から 要素数(Data)-1 まで 1 ずつ増やしながら繰り返す:
｜ もし Data[i] == target ならば:
｜ ｜ position = i
⎿ ⎿ break

もし position == -1 ならば:
｜ 表示する("見つかりませんでした")
そうでなければ:
⎿ 表示する("位置 " + 文字列(position) + " にあります")
```
