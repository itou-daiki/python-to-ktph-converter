# 線形探索

配列を先頭から順に検索するアルゴリズム

```
# 線形探索の例
data ← [2, 3, 4, 10, 40]
"検索する値を入力してください" を出力する
x ← 整数値を入力する

# 線形探索
found ← False
i を 0 から len(data)-1 まで繰り返す
    もし data[i] == x ならば
        "値", x, "は位置", i, "にあります" を出力する
        found ← True
        繰り返しを終了する

もし found == False ならば
    "値", x, "は見つかりませんでした" を出力する
```
