# 回答の度数分布

難易度: 発展
学習ポイント: 度数分布、添字変換、カウンタ配列

```
# 1から5までの回答数を数える
Responses = [1, 3, 2, 5, 3, 1, 4, 3, 2, 5, 3]
Counts = [0, 0, 0, 0, 0]

i を 0 から 要素数(Responses)-1 まで 1 ずつ増やしながら繰り返す:
｜ answer = Responses[i]
⎿ Counts[answer - 1] = Counts[answer - 1] + 1

i を 0 から 要素数(Counts)-1 まで 1 ずつ増やしながら繰り返す:
⎿ 表示する(文字列(i + 1) + "の回答数: " + 文字列(Counts[i]))
```
