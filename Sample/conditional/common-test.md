# 得点の判定

難易度: 基礎
学習ポイント: if、elif、else、論理演算

```
# 0点から100点までの得点を判定
score = 【外部からの入力】

もし score < 0 or score > 100 ならば:
｜ 表示する("0から100までの点数を入力してください")
そうでなくもし score >= 90 ならば:
｜ 表示する("評価はAです")
そうでなくもし score >= 70 ならば:
｜ 表示する("評価はBです")
そうでなくもし score >= 60 ならば:
｜ 表示する("評価はCです")
そうでなければ:
⎿ 表示する("もう一度復習しましょう")
```
