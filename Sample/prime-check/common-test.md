# 素数判定

効率的な素数判定アルゴリズム

```
# 素数判定の例
関数 is_prime(n)
    もし n <= 1 ならば
        False を返す
    もし n <= 3 ならば
        True を返す
    もし n を 2 で割った余りが 0 または n を 3 で割った余りが 0 ならば
        False を返す
    
    i ← 5
    i * i <= n の間
        もし n を i で割った余りが 0 または n を (i + 2) で割った余りが 0 ならば
            False を返す
        i ← i + 6
    True を返す

"素数かどうか調べる数を入力してください" を出力する
num ← 整数値を入力する

もし is_prime(num) ならば
    num, "は素数です" を出力する
そうでなければ
    num, "は素数ではありません" を出力する
```
