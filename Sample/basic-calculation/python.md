# 基本計算

入力と出力、四則演算の基本

```python
# 基本的な計算のサンプル
print("二つの数を入力してください")
a = int(input("1つ目の数: "))
b = int(input("2つ目の数: "))

print("a = " + str(a) + ", b = " + str(b))
print("和: " + str(a + b))
print("差: " + str(a - b))
print("積: " + str(a * b))
print("商: " + str(a // b))
print("余り: " + str(a % b))
```
