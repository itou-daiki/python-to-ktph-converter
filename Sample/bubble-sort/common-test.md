# バブルソート

隣接する要素を比較して交換を繰り返すソートアルゴリズム

```
# バブルソートの例
data ← [64, 34, 25, 12, 22, 11, 90]
n ← len(data)

"ソート前のデータ:" を出力する
i を 0 から n-1 まで繰り返す
    data[i] を出力する（改行なし）
改行を出力する

# バブルソート
i を 0 から n-1 まで繰り返す
    j を 0 から n-i-2 まで繰り返す
        もし data[j] > data[j + 1] ならば
            # 要素を交換
            temp ← data[j]
            data[j] ← data[j + 1]
            data[j + 1] ← temp

"ソート後のデータ:" を出力する
i を 0 から n-1 まで繰り返す
    data[i] を出力する（改行なし）
改行を出力する
```
